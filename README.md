# Maroune's Portfolio Website

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Overview

This portfolio website showcases projects, skills, and contact information. It's designed to be hosted on GitHub Pages.

## Files

| File | Description |
|------|-------------|
| [`index.html`](index.html) | Main HTML page with the portfolio content |
| [`style.css`](style.css) | Stylesheet for the portfolio |
| [`script.js`](script.js) | JavaScript for interactive features |
| [`profile.jpg`](profile.jpg) | Profile photo |
| [`create_portfolio_advanced.sh`](create_portfolio_advanced.sh) | Setup script for Termux |

## Getting Started

### Running Locally

To preview the site locally, you can use Python's built-in HTTP server:

```bash
cd marounebellouqid.github.io
python3 -m http.server 8080
```

Then open your browser to `http://localhost:8080`.

### Using the Setup Script

If you're using Termux on Android, you can use the included setup script:

```bash
chmod +x create_portfolio_advanced.sh
./create_portfolio_advanced.sh
```

## Customization

To personalize this portfolio:

1. Edit `index.html` to update your name, bio, and projects
2. Replace `profile.jpg` with your own photo
3. Modify `style.css` to change colors and styling
